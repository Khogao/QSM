## ✅ ĐÃ FIX - Hỗ trợ nhiều file cùng lúc

### Cách dùng:

**Kéo thả NHIỀU file cùng lúc:**
```
>>> [Kéo thả 2-3 file vào đây]
   [+] file1.jpg
   [+] file2.jpg  
   [+] file3.jpg
>>> done
```

Script đã được update để:
- ✅ Tự động tách nhiều file khi drag & drop
- ✅ Nhận diện chuỗi path có dấu ngoặc kép
- ✅ Xử lý từng file riêng biệt

**Thử lại ngay:**
1. Click `ocr-quick.bat`
2. Kéo thả NHIỀU file ảnh vào
3. Gõ `done`
4. Xem kết quả!
